 # MockGenius
Chrome Extension for mocking data, based on Vite 4 and Chrome Extension Manifest V3.

## Installation
Run：
```
npm install
```
Or
```
yarn
```

## Usage: Development Environment

Run：
```
npm run dev
```
Or
```
yarn dev
```

## Usage 
 
Run：
```
npm run build
```
Or
```
yarn build
```
